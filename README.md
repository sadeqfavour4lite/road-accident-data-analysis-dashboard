# Road Accident Data Analysis Dashboard

## Project Overview

This project presents an interactive Excel dashboard developed to analyse road accident data and identify factors contributing to casualty severity. The dashboard provides data-driven insights to support road safety improvements and decision-making.

The analysis focuses on:

* Casualty severity trends
* Vehicle involvement
* Road type analysis
* Surface conditions
* Light conditions
* Urban vs rural accident distribution
* Monthly casualty trends

## Key Insights

* Total casualties recorded: 417,883
* Cars accounted for 79.8% of casualties
* Single carriageways recorded the highest casualty rates
* Most accidents occurred during daylight conditions
* Dry road surfaces contributed to the majority of accidents
